# ✝ Futures

A minimal, static stock tracker. Live prices, multiple accounts, no backend — data never leaves your browser.

**→ [futures.brandon-knight.com](https://futures.brandon-knight.com)**

---

Built with vanilla HTML/CSS/JS. Prices via [Finnhub](https://finnhub.io). Hosted on GitHub Pages.

> To run your own version, swap `FINNHUB_KEY` in `index.html` with a free key from [finnhub.io](https://finnhub.io).
