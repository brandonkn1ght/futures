# futures

A minimal portfolio tracker. Static site, vanilla HTML/CSS/JS, no build step.

Portfolio data (accounts, holdings, history) lives in your browser via `localStorage`. Prices come from [Finnhub](https://finnhub.io).

## Setup

1. Get a free key at [finnhub.io](https://finnhub.io/).
2. Open the site, click the cog → paste the key into "Finnhub key".
3. Add accounts and holdings. Done.

The key is stored in your browser only. It is never committed to the repo.
